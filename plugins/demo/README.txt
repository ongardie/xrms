This is a simple plugin for including an MRTG page in XRMS.
This plugin can be easily modified to include any other PHP page.

Edit the following line in mrtg.php to point to your MRTG page:
        <?php include("../../../mrtg/index.php");  // Edit this to point to
your MRTG page ?>
