This is a simple plugin for including a wx200d page in XRMS.

"wx200d is a weather station data collector and server daemon for the WX200 /
WM918 / WMR918 / WMR968 weather station hardware. wx200d can log weather data
to files and/or a database."

This plugin can be easily modified to include any other system call.

Download the lastest version of wx200d from http://wx200d.sourceforge.net/
and follow the install instructions.
