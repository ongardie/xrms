TeamNotice Plugin

This plugin is used to show notices for your team when they login on the front page of XRMS.
It can be used to show informtaiton such as;

1. New products
2. Special deals
3. Staff notices

To install team notice, simple execute the sql teamnotice.sql on your database, and install the plugin in the admin menu.
